Project Name: StudentManagementSystem


This project allows the user to:
--> Add student
--> View all students
--> Search student by ID
--> Delete student

StudentManagementSystem
│
├── Main.java
├── Student.java
└── StudentDatabase.java



1. Save the files in a folder named StudentManagementSystem.
2. Compile all .java files:
# javac *.java
3. Run the program:
# java Main
